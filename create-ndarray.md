## Create N-Dimensional Array

Create 1-dimensional array

```
import numpy as np

def test_run():
    print np.array([1, 2, 3])

if __name__ == "__main__":
    test_run()
```

Here is the ouput. 

```
[1 2 3]
```

Create 2-dimensional array

```
import numpy as np

def test_run():
    print np.array([(1, 2, 3), (2, 3, 4)])

if __name__ == "__main__":
    test_run()
```

Here is the output. 

```
[[1 2 3]
 [2 3 4]]
```
